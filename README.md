### 组织介绍
这个组织是 logit 日志库的扩展仓库，包括配置文件的支持等，后续可能会考虑将 logit 主库也迁移进来，不过暂时还没有动手。

更多信息参见 logit 主仓库 [Gitee](https://gitee.com/FishGoddess/logit) / [GitHub](https://github.com/FishGoddess/logit)。
